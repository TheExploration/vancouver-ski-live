About

This site provides live status updates for Cypress, Grouse, and Mt. Seymour, using data scraped from their official websites. The site also provides an analysis of price and accessibility overview and other relevant information for a skiing trip.

Please feel free to create a new issue on the GitHub repository if you notice any bugs/errors or would like to suggest new features. This website was created by jg585Username and is hosted on https://jg585.github.io/.
